Read and follow the instructions in `skills/SKILL.md` to add a new entry to the MSI Issue Tracker Excel file.

The user's input after `/risk` is the issue description. Parse it for as much detail as possible, interview for missing fields, and write to the tracker.

If no input is provided after `/risk`, ask the user what they want to add.
